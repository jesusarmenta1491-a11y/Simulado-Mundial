# Simulador Inteligente Mundial 2026

### Descripción del Proyecto
Herramienta analítica diseñada para predecir los resultados de la Copa Mundial 2026. La aplicación utiliza modelos de **Regresión de Poisson** para estimar la probabilidad de eventos (goles), permitiendo calcular las probabilidades de victoria, empate y derrota para cada encuentro.

### Resumen Estadístico
Este modelo se basa en el análisis de datos históricos para generar proyecciones:
*   **Fundamento Matemático**: Implementación de regresión estadística de Poisson sobre variables de rendimiento deportivo.
*   **Variables de Entrada**: Análisis de goles a favor, goles en contra y fuerza ofensiva/defensiva de cada selección nacional participante.
*   **Precisión**: El modelo procesa enfrentamientos específicos para arrojar probabilidades porcentuales de resultado.

### Tecnologías Utilizadas
*   **Python**: Lógica principal y procesamiento de datos.
*   **Scikit-Learn**: Procesamiento de modelos estadísticos.
*   **Streamlit**: Interfaz web interactiva para el despliegue del simulador.
